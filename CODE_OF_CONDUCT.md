# Code of Conduct

This project follows the Contributor Covenant v2.0. All participants are expected to be respectful and professional.

Expected behavior
- Be welcoming and patient.
- Use inclusive language.
- Show respect for differing viewpoints and experiences.

Unacceptable behavior
- Harassment or discrimination of any kind.
- Personal attacks, insults, or targeting individuals.

Reporting
- If you experience or witness unacceptable behavior, please report it privately to `security@nicholashomyk.dev`.

Consequences
- Maintainers may remove or block contributors who violate the code of conduct.
